Privacy Policy

Effective Date: June 28, 2025

This privacy policy applies to the GPT integration that interfaces with your GitHub repositories to assist with software development tasks, including managing project files like /backend/task_log.md and /backend/backlog.md.

What the GPT Does

  • Reads content from specified GitHub repositories you authorize.
  • Writes to files such as task_log.md and backlog.md based on your requests.
  • Lists repository contents and file metadata to assist with task tracking and Codex handoffs.

What the GPT Does Not Do

  • It does not access private user information, emails, or commit history.
  • It does not transmit, store, or retain any data beyond your current session.
  • It does not make API calls without explicit user authorization and context.

Data Handling

All GitHub API calls are performed on your behalf after OAuth authentication. The GPT does not persist any tokens or data — all operations are ephemeral within the active session.

OAuth and Scopes

The GPT uses GitHub OAuth for secure access. Required scopes:

  • repo – for reading and writing to your GitHub files
  • read:user – to confirm identity of authenticated user
  • write:repo_hook – (optional) for automation with webhooks

Questions?

Contact the developer via your GitHub account or repository discussions.
